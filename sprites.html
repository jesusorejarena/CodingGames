Editores gráficos

Un editor de gráficos rasterizados o editor de imágenes rasterizadas es una aplicación que permite al usuario crear y editar imágenes de gráficos rasterizados de forma interactiva y almacenarlas en la computadora en un formato de archivo gráfico, como JPEG, jpg, GIF y TIFF.

Para ver imágenes, generalmente es preferible usar un visor de imágenes en vez de un editor de gráficos rasterizados.

Algunos editores están diseñados específicamente para la edición de imágenes fotorrealísticas, como el popular Adobe Photoshop, mientras que otros están más orientados a las ilustraciones artísticas, como Adobe Illustrator.

Software libre:

  • Inkscape
  • GIMP
  • Pyxel Edit
  • Piskel
